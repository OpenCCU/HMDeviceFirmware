# HomeMatic / Homematic IP Device Firmware Archive

https://openccu.github.io/HMDeviceFirmware/

This repository provides an archive of past and current firmware update files for a wide range of
HomeMatic and Homematic IP devices used with the eQ-3 CCU smart home platform.
